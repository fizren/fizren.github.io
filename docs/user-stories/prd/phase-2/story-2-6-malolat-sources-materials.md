# Story 2.6: Read the Małolat page with clear sources and materials

**Priority**: Must Have  
**Dependencies**: Story 1.4 (baseline Małolat reference exists)  
**Estimated Complexity**: Medium  
**INVEST Check**: ✓ Independent, ✓ Negotiable, ✓ Valuable, ✓ Estimable, ✓ Small, ✓ Testable

**As a** visitor researching Małolat (Zbigniew Czyżewski)  
**I want to** see claims, sources, and supporting materials in a clear structure  
**So that** I can trust and reuse the information

## Acceptance Criteria

### Scenario 1: User differentiates summary from evidence
**Given** I am reading the Małolat page  
**When** I reach a key factual section  
**Then** I can distinguish factual summary text from source references  
**And** source references are grouped in a predictable location

### Scenario 2: User can follow a source trail
**Given** the page mentions a date and place  
**When** I check references  
**Then** I find at least one clearly labeled source item  
**And** I can understand what type of material it is (document, photo, archive note)

### Scenario 3: User is not overwhelmed by archival complexity
**Given** I am a non-specialist reader  
**When** I use the page as a quick reference  
**Then** I can still extract core facts without reading all source notes  
**And** source details remain available for deeper reading

## Technical Notes

- Prefer clear sectioning over dense annotation styles.
- Keep source labeling human-readable.
- Preserve page readability as source list grows.

## Validation

- [ ] Claims and references are visually and structurally distinct.
- [ ] Source/material labels are understandable to general readers.
- [ ] Quick-read and deep-read modes both remain possible.
- [ ] Page remains coherent with additional reference entries.
