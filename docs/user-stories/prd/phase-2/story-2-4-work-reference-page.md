# Story 2.4: Keep work information stable and easy to scan

**Priority**: Should Have  
**Dependencies**: Story 1.2 (Work navigation exists)  
**Estimated Complexity**: Low  
**INVEST Check**: ✓ Independent, ✓ Negotiable, ✓ Valuable, ✓ Estimable, ✓ Small, ✓ Testable

**As a** colleague or professional visitor  
**I want to** read a stable work reference page  
**So that** I can quickly understand Piotr's professional background

## Acceptance Criteria

### Scenario 1: Visitor scans key work information quickly
**Given** I open the Work page  
**When** I scan the first screen  
**Then** I can identify role, focus areas, and how to continue reading  
**And** the page structure supports quick scanning

### Scenario 2: Returning visitor sees stable information architecture
**Given** I revisit the Work page one month later  
**When** I compare structure mentally  
**Then** section organization remains familiar  
**And** updates do not force relearning the page layout

### Scenario 3: Professional visitor avoids unrelated clutter
**Given** I only need work-related information  
**When** I read the Work page  
**Then** I am not distracted by unrelated archive content  
**And** I can leave with clear professional context

## Technical Notes

- Keep this section as a stable reference, not a feed.
- Prefer concise structure over long narrative blocks.
- Ensure readability and navigation parity on mobile.
